Introduction

This is an experimental, interim re-implementation of the Perseus Catalog. It is a web application developed atop an eXist-db XML database and its HTML templating framework. The underlying XML database contains the MODS and MADS records maintained in Perseus's catalog GitHub repository, as well as XML representations of the CITE tables previously managed by an earlier version of the catalog.

The design of this implementation is deliberately spare. There is no javascript and little CSS. It is trim, fast, and unpretentious, and it displays well in text-only browsers.
